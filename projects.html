Notable Projects

IBM Lenovo-PC Blockchain Portal

A blockchain-based billing portal that settled billing disputes between IBM and Lenovo with immutable, transparent transaction storage. Used Python extensively for data processing and process automation, with microservices deployed on IBM Bluemix.

Technologies: Python, Blockchain, Hyperledger, Node.js, Kubernetes, Docker, IBM Bluemix

Cross-Currency Payment Application

An Android application built with Cordova that used the Ripple platform to perform cross-currency and cross-border payments in seconds. Developed as a proof of concept while reviewing competing blockchain technologies.

Technologies: Cordova/PhoneGap, Ripple, Android, JavaScript

Government of Goa Web Portals

State department portals including Goa Dental College and Goa Printing Press, built on the LAMP stack and deployed on Government of Goa servers. Included security audits for compliance with Government of India norms.

Technologies: PHP, MySQL, Joomla, CodeIgniter, WordPress, HTML/CSS, JavaScript

Electronic Health Record (EHR) Management

Master's advanced project enabling secure sharing and access control of medical documents between patients and doctors/hospitals using Hyperledger Fabric, with a focus on fast, permissioned electronic exchange.

Technologies: Hyperledger Fabric, Blockchain, Computer Science Capstone
